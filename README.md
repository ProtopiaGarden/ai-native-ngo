# AI-Native NGO

Public entry point to the AI-native NGO concept, white paper, translations, PDFs, and future open infrastructure developed by [Protopia Garden](https://www.protopia.garden).

**An AI-native NGO** is a new type of cognitively augmented social institution in which humans and agent systems work as a unified cognitive system — not an NGO with a chatbot, but an organizational form proportionate to the complexity of contemporary social problems.
